<![CDATA[Inheritance Group]]> The inheritance group is a content element to reference a group of another content elements. It extends the functionality of the element group by passing on the content element configuration down the navigation line. In each navigation level the user can define per drag & drop the elements, which should be referenced by the inheritance group in the similar way the element group is edited. The content of the group is inherited automatically to the below navigation levels. Additionally the elements referenced by the inheritance group can be added, removed or rearranged in every navigation level. The new configuration will be inherited only to the below levels.
