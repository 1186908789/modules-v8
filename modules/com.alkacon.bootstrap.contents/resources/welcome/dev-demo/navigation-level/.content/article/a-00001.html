<![CDATA[Navigation Demo]]> Use Case: A list of pages, e.g. different products, should be displayed in the navigation under "Products". When user clicks on "Products" the first product is directly displayed and marked as selected in the navigation. There is no product overview page.

 

Since OpenCms 8.5.0 there is a new function page "Navigation Level" to define a navigation entry point, which automatically links to the first sub page. Use this function page, if you would like to list several pages in the navigation without an overview page.
